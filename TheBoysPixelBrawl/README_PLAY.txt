Double-click TheBoysPixelBrawl.exe to start the game.

Version: v0.6

Controls:
- WASD or Arrow Keys: Move
- J / K / L / U: Light / Heavy / Skill / Ultimate
- Campaign mode: Space to jump, Shift or Ctrl to dodge
- F11 or Alt+Enter: Fullscreen
- ESC: Pause; from pause, ESC / Q / Backspace returns to main menu

If Windows shows a security warning, choose More info, then Run anyway.
